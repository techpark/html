Подключений стилей

            <link rel="stylesheet" href="имя файла">
            <style>
                /* объявление стилей */
                @import "имя файла";
                @import url("имя файла");
            </style>

Демонстрация.

Базовый синтаксис CSS

            селектор {
                свойство1: значение1; /* комментарий */
                свойство2: значение2;
            }

Типы значений

            url("http://mail.ru/") /* строки и адреса */
            0.7 /* числа */
            90% /* проценты */
            #ff0000 /* цвет */
            rgb(255, 0, 0) /* цвет */
            uppercase /* ключевые слова */

Относительные единицы

            24px /* пиксели */
            2em /* масштабируемая */
            150% /* проценты */
            3ex /* высота символа x */

Базовые селекторы

            тег {} /* <тег></тег> */
            тег.класс {} /* <тег class="класс"></тег> */
            .класс {}
            .класс1.класс2 {} /* <тег class="класс1 класс2"></тег> */
            #идентификатор {} /* <тег id="идентификатор"></тег> */

Демонстрация.

Cелекторы атрибутов

            [атрибут] {} /* <тег атрибут></тег> */
            селектор[атрибут]{}
            [атрибут="значение"] {} /* <тег атрибут="значение"></тег> */
            селектор[атрибут1="значение1"][атрибут2="значение2"] {}

Демонстрация.

Связь селекторов

            селектор1 селектор2 {} /* <тег1> <тег2></тег2> </тег1> */
            селектор1 > селектор2 {}
            селектор1 + селектор2 {} /* <тег1> </тег1> <тег2></тег2> */

Псевдо-классы

            /* селектор:псевдокласс {} */
